## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Namrata0407/getdots-assignment.git
   cd dots-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit:
   ```
   http://localhost:5173
   ```
